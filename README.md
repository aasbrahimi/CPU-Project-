# CPU Project

# Description
This project was developed as part of a Bachelor degree in Computer Science.
It focuses on CPU architecture and low-level system concepts.

# Objectives
- Understand how a CPU works internally
- Study instruction execution and control flow
- Apply low-level programming concepts
- Analyze the interaction between hardware and software

# Technical concepts
- CPU architecture
- Instruction cycle (fetch, decode, execute)
- Registers and memory
- Control unit and ALU
- Low-level system behavior
